# Lung-Cancer-Detection
Lung Cancer Detection System

A Deep Learning based web application that detects the presence of lung cancer from CT scan images using a trained Convolutional Neural Network (CNN).
The system allows users to upload lung scan images and receive AI-powered predictions with confidence scores.

Project Overview
Lung cancer is one of the leading causes of death worldwide. Early detection can significantly improve survival rates.

This project provides an AI-assisted diagnostic tool that analyzes lung CT scan images and predicts whether the scan indicates:
Cancer Detected  Non-Cancerous

The system is built with a deep learning model trained on lung cancer image datasets and deployed through a user-friendly web interface.

Features
Upload lung CT scan images
AI-based cancer prediction
Prediction probability chart
Confidence score visualization
Download AI medical report
Scan history tracking
Clean medical dashboard UI
